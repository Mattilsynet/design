## Kode
- Bruk elementet `<mtds-chart>` med `data-variant="bar | bar-stacked | column | column-stacked | line | area | doughnut"`
- Sett inn en tabell som inneholder:
  - Første rad med grupperings-ledetekster
  - Andre rad og utover med dataserier, hvor første kolonne er dataserie-tittel
  - **Merk:** Bruk av `thead`, `tbody` eller `th` er valgfritt - radene parses uansett
  - **React?** Du kan alternativt sende inn data som en array via `data`-prop
<Canvas of={stories.Default} />

## Stående stolpediagram
- Bruk `data-variant="bar"` for stående søylediagram
- Bruk `data-variant="bar-stacked"` for stablet gruppering
<Grid data-items="300"><Canvas of={stories.Bar} /><Canvas of={stories.BarStacked} /></Grid>

## Liggende stolpediagram
- Bruk `data-variant="column"` for liggende søylediagram
- Bruk `data-variant="column-stacked"` for stablet gruppering
<Grid data-items="300"><Canvas of={stories.Column} /><Canvas of={stories.ColumnStacked} /></Grid>

## Linjediagram
- Bruk `data-variant="line"` for linjediagram eller  `data-variant="area"` for fylt linjediagram
- Bruk `data-variant="line-SMOOTHING` eller `data-variant="area-SMOOTHING"`, hvor `SMOOTHING` er et tall, for å endre hvor mye kurven skal jevnes ut (`10` er default)
<Grid data-items="300"><Canvas of={stories.Line} /><Canvas of={stories.Area} /></Grid>

## Kakediagram
- Bruk `data-variant="pie"` eller `data-variant="doughnut"` for Kakediagram
<Grid data-items="300"><Canvas of={stories.Pie} /><Canvas of={stories.Doughnut} /></Grid>

<CssVariables component='chart' />

Du kan med andre ord styre hvilket dataset som har hvilken farge, ved å f.eks gjøre:

```css
.my-chart {
  --mtdsc-chart-color-1: var(--mtdsc-chart-color-success);
  --mtdsc-chart-color-2: var(--mtdsc-chart-color-warning);
  --mtdsc-chart-color-3: var(--mtdsc-chart-color-danger);
}
```
