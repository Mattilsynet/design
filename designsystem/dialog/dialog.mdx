# Dialog

- Legg klassen `dialog` på `<dialog>`
- Åpne med metoden [`.showModal()`](https://developer.mozilla.org/en-US/docs/Web/API/HTMLDialogElement/showModal)

## Lukke knapp
- Legg `data-command="close"` på en button inni for lukke-knapp
- Første tomme knapp i `<dialog>` vil automatisk styles som lukke-knapp
- Legg evt. til flere knapper for lukking

## Klikk utenfor dialog for å lukke
